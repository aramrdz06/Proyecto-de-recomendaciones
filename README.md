# 🎬 CineBot - Sistema de Recomendación de Películas

Este proyecto es un sistema inteligente que recomienda películas utilizando la API de TMDB. Desarrollado para la clase de **Inteligencia Artificial**.

## 📌 Requisito Indispensable: API Key
Por seguridad, este código **no incluye** la llave de acceso. Para que el proyecto funcione, debes obtener una propia:
1. Regístrate en [The Movie Database (TMDB)](https://www.themoviedb.org/).
2. Ve a tu perfil > Configuración > API y genera una **API Key (v3 auth)**.
